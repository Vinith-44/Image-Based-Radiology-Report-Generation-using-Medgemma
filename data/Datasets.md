# Datasets

Raw images are not stored in this repository due to size and privacy (HIPAA) constraints.

## Datasets Used

| Dataset | Modality | Access |
|---------|----------|--------|
| Indiana University CXR | X-Ray | [🔗 Kaggle](https://www.kaggle.com/datasets/raddar/chest-xrays-indiana-university) |
| MIMIC-CXR | X-Ray | [🔗 PhysioNet](https://physionet.org/content/mimic-cxr/2.1.0/) |
| NIH ChestX-ray14 | X-Ray | [🔗 Kaggle](https://www.kaggle.com/datasets/nih-chest-xrays/data) |
| CheXpert-v1.0-small | X-Ray | [🔗 Kaggle](https://www.kaggle.com/datasets/ashery/chexpert) |
| RexGradient-160K | X-Ray | [🔗 HuggingFace](https://huggingface.co/datasets/rajpurkarlab/ReXGradient-160K) |
| CT-RATE | CT | [🔗 HuggingFace](https://huggingface.co/datasets/ibrahimhamamci/CT-RATE) |
| Brain Stroke CT | CT | [🔗 Kaggle](https://www.kaggle.com/datasets/ozguraslank/brain-stroke-ct-dataset) |
| Augmented Alzheimer MRI | MRI | [🔗 Kaggle](https://www.kaggle.com/datasets/uraninjo/augmented-alzheimer-mri-dataset) |
| Brain Tumor MRI | MRI | [🔗 Kaggle](https://www.kaggle.com/datasets/jarvisgroot/brain-tumor-classification-mri-images) |
